# Tilelayer Tester

Test XYZ tilelayer URLs to see if they are both valid and online.  Enforces presence of the `{z}`, `{x}`, and `{y}` variables in the URL.  Throws an error if the URL is not valid, or is not loading.

Preview a layer, or test it out without having to build a project or use an online code editor.

Built with [Solid](https://docs.solidjs.com), [Leaflet](https://leafletjs.com) and [Tailwind CSS](https://tailwindcss.com).
